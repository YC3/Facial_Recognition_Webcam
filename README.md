# Facial Recognition with webCam

Used pretrained models from OpenCV and dlib.

## Getting Started

### Step 1:
Download dlib facial landmark model (shape_predictor_68_face_landmarks.dat) here: http://dlib.net/files/

### Step 2:
Unzip the file, get your webcam ready and run:
```Python
python3 ./main.py
```
